#  Task 3: Web Application for Student Performance Prediction

Task 3 involved developing a web-based application using Flask to predict student performance based on various input parameters. 
The objective of this task was to integrate machine learning with web development, enabling users to interact with the model through a user-friendly interface.

The dataset used for this task contained information about students, such as hours studied, attendance, sleep hours, previous scores, and tutoring sessions. 
These features were selected as input variables to predict whether a student would pass or fail. The dataset was preprocessed using Pandas, 
where missing values were handled and relevant features were selected.

A machine learning model, specifically a Random Forest Classifier, was trained on the dataset.
Random Forest is an ensemble learning technique that combines multiple decision trees to improve prediction accuracy. 
The model was trained and saved using joblib so that it could be reused in the web application without retraining.

The Flask framework was used to build the web application. Flask is a lightweight web framework in Python that allows easy integration of backend logic with frontend design. 
The application consisted of routes for rendering the homepage and handling prediction requests.

The frontend was designed using HTML, CSS, and JavaScript. Input fields were provided for users to enter student details, and icons were added to improve the user 
interface. JavaScript (Fetch API) was used to send data asynchronously to the Flask backend, enabling real-time predictions without refreshing the page.

When the user enters details and clicks the predict button, the input data is sent to the backend, where it is processed and passed to the trained model. 
The model returns a prediction, which is displayed on the webpage as either “Pass” or “Fail.” Additional features such as a reset button were added to enhance 
usability.

This task demonstrated the practical application of machine learning in real-world scenarios by deploying a model as a web application. 
It also improved skills in full-stack development, including backend programming and frontend design.

Overall, Task 3 provided valuable experience in integrating machine learning with web technologies, making it possible to deliver intelligent systems 
that users can interact with easily.
